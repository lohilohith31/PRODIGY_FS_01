# 🔐 Secure User Authentication System

This project is a backend authentication system developed using Spring Boot. It provides secure user registration and login functionality using JWT (JSON Web Token).

## 🚀 Features
- User Registration
- Secure Login with Password Encryption (BCrypt)
- JWT Token Generation
- Protected API Routes
- Exception Handling

## 🛠️ Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- H2 Database

## 📌 API Endpoints

### Register
POST /auth/register

### Login
POST /auth/login

### User Dashboard
GET /auth/user/dashboard

## 🔐 Security
- Passwords are encrypted using BCrypt
- JWT is used for authentication and authorization

## 📸 Output
- Successfully registers user
- Generates JWT token on login
- Allows access to protected routes with token
